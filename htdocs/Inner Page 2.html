What will I do?

There's still so much content to be added, so many new characters I want, I don't think I'll be quitting this game any time soon. The
game's story has yet to conclude and I'd like to see it through. In fact, this has been the first game I've spent money on, so I'd feel
like that would be wasted if I stopped. Even if I did quit, this game has hooked me back before and I'm certain it would again.

Conclusion

Although someday I might fall out of love of this game, the good memories I made playing this game will always stay within my mind.
